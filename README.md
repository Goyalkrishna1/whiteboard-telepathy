# 🎨 Whiteboard Telepathy

A real-time collaborative whiteboard application where multiple users can draw together, save their canvases, and collaborate seamlessly.



## 🎥 Project Video
https://github.com/user-attachments/assets/abdb8a61-3dd2-4c27-b48e-ee0b99a462d4


## 🚀 Features

- ✏️ Freehand Drawing
- 🎨 Multiple Colors
- 📏 Adjustable Brush Size
- 🧹 Eraser Tool
- ↩️ Undo / Redo
- 🗑️ Clear Canvas
- 💾 Save Canvas
- 🔐 User Authentication (JWT)
- 👥 Multi-user Support
- ⚡ Real-time Updates using Socket.io

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS Modules
- Axios
- React Router
- Perfect Freehand
- Socket.io Client

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Socket.io
